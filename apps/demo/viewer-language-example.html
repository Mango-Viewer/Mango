Mango Viewer - Language Configuration Examples

Example 1: Spanish UI (es)

This viewer is configured with Spanish language:

<mango-viewer
  manifest-id="..."
  config='{"language":"es"}'
></mango-viewer>

Example 2: English UI (en) - Explicit

Explicitly set to English:

<mango-viewer
  manifest-id="..."
  config='{"language":"en"}'
></mango-viewer>

Example 3: No Language Config (Browser Default)

When no language is specified, the viewer uses the browser's language (navigator.language), falling back to English if unavailable:

<mango-viewer
  manifest-id="..."
></mango-viewer>

How Language Resolution Works

The viewer resolves the UI language in this priority order:

  1. Config language (highest priority): config='{"language":"es"}'
  2. Browser language: Falls back to navigator.language
  3. Default: Uses 'en' if nothing else is available

Supported Language Formats

  • Simple codes: "en", "es", "fr", "de"
  • Locale variants: "en-US", "en-GB", "fr-CA", "es-MX"

Available Translations

Currently, the viewer has translations for:

  • en - English (default)
  • es - Spanish (Español)

Other language codes will fall back to English.
